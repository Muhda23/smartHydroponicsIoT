## Features
- MQTT communication
- Real-time dashboard monitoring
- Historical Data Monitoring with Graphic visualitation
- Automatic Adjusting Nutrient
- Reset Wifi-Config (So, can use other WiFi if this tools placed on other place)

## Hardware
- Microcontroller: ESP32
- Sensor: Ultrasonic level sensor (HCSR04) ==> to measure volume hydroponic media
	  GravityTDS meter ==> to measure nutrient concentrate (ppm)
	  DS18B20 Themperature Sensor ==> to measure themperature hydroponic media
- Actuator: Pump, relay & LED

- MQTT Broker (emqx)
